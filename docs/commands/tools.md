# Tools

### Sauce

A command for searching a source from an image using saucenao.

> /sauce \<attachment>

| Arguments     | Description                                        |
| ------------- | -------------------------------------------------- |
| \<attachment> | Discord attachment. Please use `.png` or `.jp[e]g` |
